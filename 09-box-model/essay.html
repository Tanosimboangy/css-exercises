CSS exercise 09: The Box Model

1- A short essay on what the Box Model is

I am writing this essay to explain what box model is. The box model is a container that contains every element in css. It helps the user to displays the layout in different styles or complexities. There are two types of boxes. For block box, the box will extend itself to give 100% enough space for its contents. For that to happen, the box will break onto new line. This has to respect the height and the width properties. However, the margin, the padding and the border control the limitation of the elements which fit inside the box and push away other elements which don't fit inside. Whereas, for inline box, the box will not break onto a new line. Moreover, the properties which all apply with block box will not apply with this sort of box. However, the margin, the padding and the border control the limitation if the elements which fit inside of the box but will not push away any elements.

3- Description each of the following keywords:

  • Content-Box: The content of the box if the place where the text or the the image or the characters appear on the layout.
  • Padding-Box: The padding-box is the space which separates the content and the border. This is transparent.
  • Border-Box: The border-box is the border that wraps the padding and the content.

4- Description each of the following properties and understanding their values:

  • Border: The border property CSS is a shorthand syntax in CSS which sets the element's border.
    • The border-width: specifies how thick you want the border to be.
    • The border-style: specifies the style of the line arround the element which the user wants it to be.
    • The border-color: specifies the color of the border.
  • Margin: The margin property CSS sets the margin area on all four sides (right, left, top, bottom) of an elemnt.
    • When one value is set, it applies to all fours sides.
    • When two values are set, the first one applies to the bottom and the second one applies top, and the third one applies to the right and the left.
    • When three values are set, the first one applies to the top, the second one applies to the right and the left, the third one applies to the bottom.
    • When four values are set, each one applies to each side (right, left, top, bottom).
  • Padding: The padding property CSS sets the pdding area of an element.
    • When one value is set, it applies to all fours sides.
    • When two values are set, the first one applies to the bottom and the second one applies to the top, and the third one applies to the right and the left.
    • When three values are set, the first one applies to the top, the second one applies to the right and the left, the third one applies to the bottom.
    • When four values are set, each one applies to each side (right, left, top, bottom).
  • Display: The display property CSS specifies how you want the box behaves.
  • Box-sizing: The box-sizing property CSS sets the size (width, height) of a box you want it to be.

Answering these questions:

  • When two differently sized margins of two different boxes collide: The one which has the bigest value will override the space.
  • The difference between display values inline, inline-block, and block.
    • For display inline: Inline elements don't start on a new line but they appear on the same line as the content. You can add width to the left and right on an inline elements, whereas you can't add height to top and bottom margin and padding of an inline element.
    • For display inline-block: Inline-block elements are similar to inline elements, because they can padding and margins. However, they are really usefull for creating a horizontal navigation menu.
    • For display block: A block element always start on a new line and fills up the space left and right. The advantage of using block elements is you can add margin and padding on all four sides.
