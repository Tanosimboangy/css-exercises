FLOAT AND BLOCK FORMATTING CONTEXTS

floats

  • Default value:
    • The default value for float is none.
    • The default value for clear is none.
  • The possible values for clear are:
    • Right
    • Left
    • None
    • Both
  • The rules associated with a floated item in terms of source order is that the flotted element must display before the other element.
  • We can float other elements besides images. But sometimes we have to choose the display value.
  • YES! There is a way to create a two-column layout with floats.

BFC (Block formatting context)

  • Description on my own words what BFC is all about.
    • According to me, BFC is a part of the viewport css displayed on a web page, that the block boxes are appeared.
  • We can clear a float by creating a block formatting context on the parent, this is achieved by putting a certain set of properties on the container. Those properties are:
    • Overflow: scroll
    • overflow: hidden
    • display: flex
    • float: left
      • display: table
