Flexbox

    • The order of property values in the shorthand flex property is flex-grow, flex-shrink and flex-basis.
    • For this example: flex: 2 1 auto;: 2 represents the value of the flex-grow, 1 represents the value of the flex-shrink and auto represents the value of the flex-basis.
  2. Before we can use align-content we need these flex-wrap or flex-flow values:
    • For flex-wrap values we need: wrap, wrap-reverse.
    • For flex-flow values we need: row wrap, column wrap-reverse, column wrap.
  3. The diffrence between align-content and align-items is:
    • Align-content only applies to the multiple-line of the flex container and it aligns the flex lines.
    • Whereas, align-items only applies to all flex container and it aligns the flex items along the cross axis of each flex items.
  4. The flex item's align-self property overrides the other. This is because in Grid, it aligns the item inside the grid area. In Flexbox, it aligns the item on the cross axis.
  5. The other commonly-used box model property which is similar to flex-basis is flex-grow.
  6. There are two ways to change the ordering of flex items within a flex container. One is done on each flex item by using the order property. The other can change the whole order of the flex items at the container level. That property is flex direction, only by using one of these values: row, column, row-reverse, column-reverse.
  7. Let's say we wanted some space between each of our three flex items in a flex column. Some possible ways to achieve this:
    • we can use margin: auto to set the space around each items.
    • Moreover, to control the space between the items we can use align-content with the values space-between or space-around.
    • The third way to achieve this is by using justify-content distributes space between and around content items.
