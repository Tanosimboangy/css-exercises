Custom Properties

  • We have to use some characters when defining a Custom Property's name. Those are:
    • Two dashes(--) preceded a selected name.
    • Dashe(-) or underscore(_) if the name is more than two words.
  • Characters we can't use in a Custom Property name are:
    • Quotation mark("") or ('').
    • comma(,).
    • doubt(.).
    • slash(/).
  • The possible values we can assign to a new Custom Property are:
    • color
    • padding
    • background-clolor
  • It is useful to create Custom Properties using the :root or html selector when we want to avoid rewrite the same values every property. To be clear the user can reduce the need of repetition.
