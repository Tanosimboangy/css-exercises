Research aboutThe Calc() function

  1. Some values that a calc() function can support are:
    • Number.
    • Key words.
    • Normal text.
    • Frequency.
    • Percentage.
    • Fraction.
  2. To control the mathematical order of operation (PEDMAS / BODMAS) in the function, so that something like calc(2 + 3 * 4) equals 20, we can put brackets round the addition part that we want to be separately calculated from the whole operation. For example calc(var((2 + 3) * 4));
  3. We can't use a calc() function to convert one type of unit to another type of unit.
  4. If we want to convert a unitless number to a unit value, what we have to do is to multiplie the value by 1(unit) of the unit we want. For example: font-size: calc(var(line-height) * 1px);
  5. The plus and minus operators inside the function need spaces around them because it's required, otherwise the calc fonction won't work just as the operator plus(+) or minus(-) doesn't work. And the number after the brackets will be negative if the spaces are missed.
  6. We can convert a value into a negative value by multipling the value by -1. For example: margin: calc(var(font-size) * -1);
  7. Calc() function can't work with colour values? e.g. a light red plus a light red should equal a darker red.
  8. You can mix unit if you use abstraction or or addition.
