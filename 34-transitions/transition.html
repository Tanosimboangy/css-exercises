RESEARCH ABOUT TRANSITION

  • The transition property is a 'shorthand property'. Other properties are included in this shorthand property. These are:
    • Transition-property.
    • Transition-duration.
    • Transition-timing-function.
    • Transition-delay.
  • The difference between a timing function, delay, and duration:
    • For timing function is used to define the speed over time of an object being transitioned from one point to another.
    • Delay specifies the duration to wait before the transition effect will start.
    • For duration specifies how long a transition effect takes to complete. It might be in seconds (s) or milliseconds (ms).

We can select more than one property to transition at a time. This is possible by adding a comma-separated list of properties within the transition-property value. To transition all specified properties over the same duration, include only one value for transition-duration . But pay attention to this, the order of the values is necessary when defining multiple transition durations
