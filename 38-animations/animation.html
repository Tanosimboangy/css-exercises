Keyframe animations

  • 1 _ Keyframe Animation setup needs some unique code, it starts with @Keyframe, that means at the beginning of the animation.
  • 2 _ What makes the keyframes code different to the rest of CSS is that you have to set @keyframes (at the beginning of the code) rule before setting up all animation properties and the name, otherwise it's not going to work.
  • 3 _ To use a keyframe animation we set up earlier as a property in a ruleset for an element, we have to define a specific name for the @keyframes with its rule and need to call that animation name to link an animation to its keyframe declaration. Moreover, we also need the animation duration.
  • 4 _ Keyframe animations have other keywords that aren't available in transitions, these keywords are:
    • Infinite
    • Reverse
    • Running
    • Pause
