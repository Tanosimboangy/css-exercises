Research about Shadows

There are two types of shadows:

  • Box Shadows, or the shadows that run around or inside a box.
  • Text Shadwos, or the ghostly shadow outlines of text.
  • Box-shadow and text-shadow have very similar value syntax. However, they have some differences like:
    • We can use thickness value with box shadow but we can not use it with text shadow.
    • We can not use inset value with the text shadow whereas we can use it with box shadow.
  • If I wanted to create a hard-edge shadow, one that looks like a solid border, the combination of values I would need to use is 0 for the X, 0 for Y, 0 for the blur but a value bigger than 0 for the thickness.
  • Yes, it is possible to create a shadow on a circular element. We can create a shadow on a polygon, but with these conditions:
    • It can only work with SVG.
    • Moreover, you have to use filter
  • Yes, it is possible to create multiple shadows on the same element.
  • We can use shadow in every case but we can not use it if not visible.
