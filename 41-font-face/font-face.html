  • There are several types of font formats, e.g. WOFF, TTF, etc.
    • Woff(Open Font Format)
    • Woff2(Web Open Font format 2.0)
    • eot(Embedded Open type)
    • otf(Open type Font)
    • ttf(True Type Font)
    • svg
  • "Woff" is one of these font file formats which is a good candidate to use in all browsers.
