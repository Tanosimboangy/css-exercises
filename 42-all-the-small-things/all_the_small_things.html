Outline

  • The outline property specifies the outline to be added to the box.
  • Medium invert color is the default value for this property.
  • This works quite similar to border but it is set on the border of the border.
  • There are
    • outline-style(specifies the style of the outline)
    • outline-color(specifies the color of the outline)
    • outline-width(specifies the width of outline)
    • outline-offset(adds space between an outline and the edge or border of an element.)
    • outline(is a shorthand property)

Filter

  • The CSS filters property are used to adjust the renderign of an image, a bacground, or a border.
  • None is the default value for this property, so when the value is invalid, the value returns to none.
  • By using this property, you can control how you want to render an image or background by using oone these values:
    1. Blur (This applies a Gaussian blur to the input image).
    2. Brightness (This applies a linear multiplier to the input image, making it appear more or less bright).
    3. Contrast (This adjusts the contrast of the input image).
    4. Drop-shadow (This applies a drop shadow effect to the input image).
    5. Grayscale (This will convert the input image to grayscale).
    6. Invert (This inverts the samples in the input image).
    7. Opacity (This applies transparency to the samples in the input image).
    8. Saturate (This saturates the input image).
    9. Sepia (This converts the input image to sepia).
    10. Brightness (This applies a linear multiplier to the input image, making it appear more or less bright).

Appearance

  • The appearance property is used to display an element using a platform-native styling based on the users’ operating system’s theme.
  • The appearance property is used for one of two reason: To apply and platform specific styling to an element that doesn’t have it by default. And To remove platform specific styling to an element that doesn’t have it by default.

:focus

  • The :focus CSS pseudo-class represents an element (such as a form input) that has received focus.
  • It is generally triggered when the user clicks or taps on an element or selects it with the keyboard's "tab" key.
  • Note: This pseudo-class applies only to the focused element itself. Use :focus-within if you want to select an element that contains a focused element.

:not()

  • The :not() CSS pseudo-class represents elements that do not match a list of selectors.
  • Because it prevents specific items from being selected, it is known as the negation pseudo-class.
  • This can increase the specificity of a rule.

:matches

  • The :is() or :matches CSS pseudo-class function takes a selector list as its argument, and selects any element that can be selected by one of the selectors in that list. This is useful for writing large selectors in a more compact form.
  • :matches can not be nested and it does not work with :not.

::first-letter

  • The ::first-letter CSS pseudo-element is used to add a style to the first letter of the specified selector..
  • font properties, color properties, background properties, margin properties, padding properties, border properties, text-decoration, vertical-align (only if float is 'none'), text-transform, line-height, float, clear.

@supports

  • The @supports CSS at-rule lets you specify declarations that depend on a browser's support for one or more specific CSS features.
  • The supports condition consists of one or more name-value pairs combined by conjunctions and, or, not.

attr()

  • The att() selector is used to select elements with a specified attribute and value.
  • It can also be used on pseudo-elements, in which case the value of the attribute on the pseudo-element's originating element is returned.
  • These are the type of its units: String, Color, Url, Number, Angle, Cap, Ch, Em, Ex, Ic, Lh, Rlh, Rem, Vb, Vi, Vw, Vh, Vmin, Vmax, Mm, Q, Cm, In, Pt, pc, px, deg, grad, rad, turn, ms, s, Hz, kHz, %.
